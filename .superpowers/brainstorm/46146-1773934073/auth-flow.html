Key Insight: What Changes from the Current Demo

Current Demo (wrong)

OAuth path: REST API with bearer token → server signs transactions

Direct path: mob with user's mnemonic → user signs & pays gas

✘ Two separate paths, neither matches abstraxion

New Design (correct)

Single path: OAuth for auth → mob generates session key → mob signs locally with authz grants

Gasless: Treasury pays all gas via fee_granter

✔ Matches abstraxion React Native exactly
