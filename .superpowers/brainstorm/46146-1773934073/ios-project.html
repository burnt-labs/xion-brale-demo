UI Framework Jetpack Compose SwiftUI
DI Hilt Manual AppContainer
Navigation Compose Navigation NavigationStack
HTTP Client Retrofit + OkHttp URLSession + Codable

mob Build Script: iOS Targets

build-mob-ios.sh key steps
# 1. Clone burnt-labs/mob → third_party/mob

# 2. Install Rust iOS targets
rustup target add aarch64-apple-ios        # Device
rustup target add aarch64-apple-ios-sim    # Simulator (Apple Silicon)
rustup target add x86_64-apple-ios         # Simulator (Intel)

# 3. Build static libraries (.a) for each target
cargo build --release --target aarch64-apple-ios
cargo build --release --target aarch64-apple-ios-sim
cargo build --release --target x86_64-apple-ios

# 4. Create universal simulator binary
lipo -create \
  target/aarch64-apple-ios-sim/release/libmob.a \
  target/x86_64-apple-ios/release/libmob.a \
  -output libmob-sim-universal.a

# 5. Generate Swift bindings via UniFFI
cargo run --bin uniffi-bindgen generate \
  --library libmob.a --language swift --out-dir bindings/
# → mob.swift, mobFFI.h, mobFFI.modulemap

# 6. Package as xcframework
xcodebuild -create-xcframework \
  -library target/aarch64-apple-ios/release/libmob.a \
  -headers bindings/ \
  -library libmob-sim-universal.a \
  -headers bindings/ \
  -output Frameworks/libmob.xcframework

iOS Implementation Phases

Phase 1
Scaffold + Build System
Xcode project, build-mob-ios.sh, xcframework integration, verify mob.swift compiles
Phase 2
Services Layer
MobSigningService, OAuthService, SecureStorage, NetworkService, PKCE util
Phase 3
Session + Repository
SessionManager (full auth flow), XionRepository, Models
Phase 4
UI (5 Screens)
Connect, Wallet, Send, Contract, History — matching Android layouts
Phase 5
Polish + Testing
Deep links, dark mode, error handling, Keychain cleanup, unit tests
